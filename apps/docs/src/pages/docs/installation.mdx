# Installation

## Quick Start

```bash
npm install ui.landing
# or
yarn add ui.landing
# or
pnpm add ui.landing
```

## Required Dependencies

```bash
npm install tailwindcss postcss autoprefixer
npm install @radix-ui/react-* @next-ui/org/react lucide-react
```

## Setup Tailwind CSS

```ts
// filepath: tailwind.config.ts
import { withLanding } from "ui.landing/config";

export default withLanding({
  content: [
    "./pages/**/*.{ts,tsx}",
    "./components/**/*.{ts,tsx}",
    "./app/**/*.{ts,tsx}",
  ],
});
```

## Add Base Styles

```css
// filepath: app/globals.css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  :root {
    --landing-radius: 0.5rem;
  }
}
```

## Configure Components

```ts
// filepath: components/ui/index.ts
export * from "ui.landing/components";
```

## Verify Installation

```tsx
import { Button } from "ui.landing";

export default function Page() {
  return <Button>I'm Working!</Button>;
}
```
